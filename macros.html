Изученные нами макросы:

  • Макрос копирования:

    • Копирует заданный текст и впоследствие позволяет мгновенно напечатать кусок текста любой длины.

    • If Worksheets(ActiveSheet.Name).Name = "Лист1"

      Then ActiveCell.FormulaR1C1 = "mindless self indulgence "

      End If

      End Sub

  • Макрос замены:

    • Заменяет все заданные элементы на другие заданные элементы во всем документе.

    • Cells.Replace What:="23", Replacement:="56", LookAt:=xlPart, SearchOrder _ :=xlByRows, MatchCase:=False, SearchFormat:=False, ReplaceFormat:=False

      End Sub

  • Макрос форматирования текста:

    • Применяет заданное форматирование к тексту.

    • Selection.Font.Bold = True Selection.Font.Italic = True

      End Sub
